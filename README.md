## ⚖️ Social Dynamics Sandbox (Frame-Gym Pro)

Social Dynamics Sandbox is an advanced LLM-based application designed to simulate and analyze social dynamics and power structures in modern text-based dating (e.g., Tinder, Bumble).

Built using Python, Streamlit, and the Google Gemini API, the application acts as both a roleplay simulator and an analytical coaching tool. It leverages deep prompt engineering to create highly specific behavioral archetypes and dynamic conversational constraints.

### 🚀 Core Features

* **🧠 AI Roleplay Engine:** Simulates real-time text interactions where the user must maintain their conversational "Frame" against an AI partner.
* **🎭 10 Psychological Archetypes:** The AI dynamically adapts its prompt to evaluate the user based on 10 specific interaction styles (e.g., The Stoic Sage, The Chad, The Data-Driven Geek, The Gentle Dom).
* **🎯 Dynamic Roles:** Switch between "Pursuer" (active seduction) and "Desired" (passive filtering) to test different relationship dynamics.
* **🦇 Adaptive CSS & UI:** Features UI toggles like the "Gothifier" that inject custom CSS and alter the AI's aesthetic output via HTML tags dynamically.
* **🕵️‍♂️ The Coach Room (Frame Analyzer):** A dedicated stateless environment where users can paste real-world chat logs. The LLM evaluates the power dynamic, scores the "Frame", identifies needy behaviors, and suggests optimal responses.
* **⚙️ Enterprise-grade Architecture:** Includes automated API load balancing (Model Roulette Fallback) to prevent ResourceExhausted errors, multi-language UI state management, and strict separation of public code and private data via Streamlit Secrets.
* **🎲 Hidden Probability Triggers:** Contains logic for probabilistic Easter Eggs (custom context injection) based on specific UI parameter combinations.
